    Changelog

    Version 1.0.0: Initial release. Getting started and whatnot.

    Version 1.1.0: Added timer functionality, image functions, and more time funtions!

    Version 1.2.0: Improved UI and added notepad feature. Made things more round and added scrolling to the main part of the file.

    Version 1.2.1: Removed content drawer and all elements and code related to it. Added this changelog so I can keep track of my updates! Also added random twitch emotes into here because I thought I should spice things up just a little bit.


    Version 1.2.5: Added right menu functionalities and reprogrammed upload buttons and inputs (both locally and URL) so they display to the correct class and ID. I also modified the body CSS code, making it so that the image you upload via URL or locally will not tile and repeat (although it is funny with some images).

    Version 1.2.7: About the creator section and a to-do section too.

    Version 1.3: Added mobile support and adaptability, with side buttons you can press to open up either side menu. Text also adapts to screen size, so it's still readable, but isn't overflowing or looks weird!

    Version 1.3.1: Made side buttons bigger and added a startup screen (because I found it funny). Screen fades away when you click, "continue."

    Version 1.3.5: Added checklist and functions, allowing you to make a checklist to keep track of what you need to do while drawing (or something).

    Version 1.3.8: Added a profile section, and made a menu to enter your username and the button text would automatically update when text is put down or deleted.

    Version 1.3.9: Made all input fields have padding and made them look nice. Also added a profile picture and a way to paste in a URL to update the picture on the top of the screen. I am still figuring out a way to add a way to upload pictures locally on the device to replace the picture, as well as a default fallback picture so that URL's or photos that are not uploaded are defaulted back to the photo. Next couple of versions, I will add features that locally save settings as cookies into the browser, so that no imputs or settings are lost (this includes images uploaded via URL).

    Version 1.4: Added a Pomodoro timer, which is a timer in which you work for a set amount of time (in this case: 25 minutes), and then after, you take a 5 minute break. You repeat the process around 3-4 times before taking a longer break (10-15 minutes). This is then repeated until the person has finished. I also added a progress text and percentage at the bottom of the checklist area so you know how much you've completed while you work.

    Version 1.4.5: Added themes to the files, using a very simple script and modifications to the background color and CSS of elements within the body itself to alter its colors and such.
